## Before you start

Ensure you have:

- <IconCheck /> Windows 11 license (Pro, Pro for Workstations, or Enterprise)
- <IconCheck /> 64-bit CPU (x64) or ARM CPU (ARM64) - Use the matching Windows ISO
- <IconCheck /> **All important data backed up**
- <IconCheck /> USB drive (only if installing from USB)

<Callout type="info" title="Note">
  Windows Home is not supported. You can upgrade to Pro later by changing your product key.
</Callout>

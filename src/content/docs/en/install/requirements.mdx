# Installation requirements

Verify that you meet all requirements before proceeding.

## System requirements

Before proceeding, ensure you have:

- **Backed up all important data** to a safe location:
  - Cloud storage (Google Drive, Microsoft OneDrive, Dropbox)
  - External storage (NAS, portable drive, USB drive)

- Met the **[Windows 11 system requirements](https://www.microsoft.com/windows/windows-11-specifications#table1)**

- **64-bit CPU (x64) or ARM CPU (ARM64)** - Use x64 Windows ISO for Intel/AMD CPUs, or ARM64 Windows ISO for ARM systems ([check your system architecture](https://support.microsoft.com/en-us/windows/which-version-of-windows-operating-system-am-i-running-628bec99-476a-2c13-5296-9dd081cdd808))

- **Supported Windows edition:** Pro, Pro for Workstations, or Enterprise (Windows Home is not supported)

<Callout type="warning" title="Important">
  The installation process reinstalls Windows, which will erase your current installation. Back up
  all important data before proceeding.
</Callout>

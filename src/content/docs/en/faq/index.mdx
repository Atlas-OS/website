## Quick answers

### What is AtlasOS?

AtlasOS is an open-source project that enhances Windows by applying privacy, usability, and performance optimizations while maintaining functionality and customizability.

### Is AtlasOS safe?

Yes, AtlasOS is open-source and community-reviewed. All modifications are transparent and available for inspection on GitHub.

### Which Windows versions are supported?

AtlasOS is designed for **Windows 11** (Pro, Pro for Workstations, or Enterprise editions).

<Callout type="warning" title="Windows Home Not Supported">
  Windows Home is not supported. You can upgrade to Pro by changing your product key.
</Callout>

# General FAQ

General questions about AtlasOS features, updates, and usage.

Keeping your AtlasOS installation current ensures you benefit from the latest features, security updates, and compatibility improvements. Update options depend on your current version.

## Checking your AtlasOS version

To check your AtlasOS version:

1. Open **Settings** <Icon name="tabler:arrow-right" class="inline h-4 w-4 mx-1" /> **System** <Icon name="tabler:arrow-right" class="inline h-4 w-4 mx-1" /> **About**
2. Look for the Windows version information
3. Check if it mentions AtlasOS or Atlas version

Alternatively, you can check the version in the Atlas folder or run system information tools.

## Performance

### Will AtlasOS improve my gaming performance?

AtlasOS includes optimizations that may improve gaming performance by reducing background processes and system overhead. Results vary by hardware and usage patterns.

### How much faster is AtlasOS?

Performance improvements vary depending on your hardware and usage patterns. Many users report:

- Reduced RAM usage
- Lower CPU overhead
- Faster boot times
- Improved gaming frame rates

### What optimizations does AtlasOS include?

AtlasOS applies various privacy, usability, and performance tweaks:

- Removes telemetry and unnecessary services
- Optimizes system processes
- Disables background tasks
- Improves resource allocation

## Compatibility

### Can I use Windows Store apps?

Yes, Windows Store apps work normally with AtlasOS.

### Will my existing software work?

Most Windows software works normally with AtlasOS. Some enterprise software may require specific Windows features that have been optimized.

### Can I use Windows Update?

Yes, Windows Update works normally. You can configure driver updates through the Atlas folder.

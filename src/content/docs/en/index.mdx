## What is AtlasOS?

AtlasOS (also called Atlas) is an open-source modification of Windows that removes unnecessary services, reduces telemetry, and optimizes system performance while maintaining full functionality. All modifications are transparent, community-reviewed, and available on GitHub.

### Key features

- **Privacy-focused**: Removes telemetry and tracking services
- **Performance-optimized**: Reduces system overhead and improves gaming performance
- **Fully customizable**: All tweaks and settings can be enabled or disabled
- **Open-source**: All code is publicly available and community-reviewed

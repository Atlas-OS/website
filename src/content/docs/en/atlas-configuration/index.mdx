# Atlas configuration

Customize your AtlasOS installation to match your preferences and needs. All configuration options are conveniently located in the Atlas folder on your Desktop or in the Start Menu, giving you full control over your system's behavior.

<Callout type="info" title="Missing Atlas Folder?">
  If you can't find the Atlas folder, check out [our FAQ guide](/docs/faq/post-install/) for
  instructions on restoring it.
</Callout>

## What you can configure

AtlasOS provides several ways to customize your installation:

- **Basic Settings** - Enable or disable Windows features like Bluetooth, Game Mode, and notifications
- **Optional Tweaks** - Personalize your interface with context menu styles, Alt-Tab appearance, and more
- **Advanced Options** - Fine-tune system behavior for power users who want detailed control
- **Security Settings** - Configure security features and privacy options
- **Windows Settings** - Quick access to common Windows configuration options

<Callout type="tip">
  Most configuration changes require a restart to take effect. You'll see a reminder when needed.
</Callout>

## Understanding the Atlas folder

The Atlas folder is located at <CodeInline text="C:\Atlas\" /> and contains all configuration files organized into logical sections:

<DirectoryTree>
C:\Atlas
├── Config
│   ├── General.ini
│   └── Advanced.ini
├── Scripts
│   └── CustomScripts.ps1
└── Logs
</DirectoryTree>

Each section of the configuration guide explains where to find specific files and what they do.

<Callout type="warning" title="Modify with Caution">
  Only modify configuration settings if you understand what they do. Incorrect changes may affect
  system stability or functionality. When in doubt, consult the documentation or ask for help in our
  [Discord community](https://discord.com/invite/atlasos).
</Callout>

## Getting started

Ready to customize your AtlasOS installation? Start with:

1. **[Atlas Folder Overview](/docs/atlas-configuration/atlas-folder/)** - Learn about the folder structure
2. **[Basic Settings](/docs/atlas-configuration/settings/)** - Enable or disable features you need
3. **[Optional Tweaks](/docs/atlas-configuration/tweaks/)** - Personalize your interface

Most settings include clear instructions on when and why you might want to use them.

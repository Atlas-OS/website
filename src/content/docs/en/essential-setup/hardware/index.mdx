# Hardware optimization

These optional optimizations can significantly improve your system's performance, especially for gaming. Hardware optimization is entirely optional but highly recommended for getting the most out of your AtlasOS installation.

<Callout type="info" title="Update BIOS First">
  We recommend updating your BIOS before enabling XMP/DOCP or Resizable BAR. If you enable these
  features before updating BIOS, your settings may revert after the update.
</Callout>

## Recommended order

For best results, follow this order:

1. **[Update BIOS](/docs/essential-setup/hardware/bios-update/)** - Do this first to avoid settings reverting
2. **[Enable XMP/DOCP](/docs/essential-setup/hardware/memory/)** - Unlock your RAM's full speed
3. **[Enable Resizable BAR](/docs/essential-setup/hardware/gpu/)** - Improve GPU performance

## What these features do

### XMP/DOCP (Memory Overclocking)

- Allows your RAM to run at its advertised speed
- Can significantly improve performance, especially in games
- Intel uses XMP, AMD uses DOCP

### Resizable BAR (GPU Enhancement)

- Improves communication between CPU and GPU
- Can improve FPS by up to 15% in some games
- Requires NVIDIA GeForce 30 series or AMD Radeon RX 6000 Series or newer

### BIOS updates

- Provides additional hardware support
- Security fixes and bug fixes
- More configuration options
- Improved performance and stability
